COMBは、些細な点でユーザーのオンラインアクティビティをサポートするDiscordボットです。コマンド操作をはじめとしたインタラ クティブな方法で、簡単かつ素早く機能を利用することができます。

## 機能
### コマンド操作
|コマンド|機能|
|---|---|
|?help|利用可能なコマンドとその使い方を参照する|
|?yd|YouTubeの動画をダウンロードする|
