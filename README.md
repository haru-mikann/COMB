COMBは様々な場面でユーザーのオンラインアクティビティをサポートするDiscordボットです。
コマンド操作などのインタラクティブな方法で簡単かつ迅速に機能を利用できます。

## 機能
### コマンド操作
|コマンド|機能|
|---|---|
|?help|利用可能なコマンドとその使い方を参照する|
|?yd|YouTubeの動画をダウンロードする|
